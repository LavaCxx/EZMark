## EZMark

一个简单易用的照片参数水印生成器。
[在线地址](https://ezmark.lavac.cc/)
![example](https://i.imgur.com/sraFwVS.png)

目前支持的图片格式包括浏览器本身能支持的格式以及苹果的`.heic`格式。

已添加以下品牌的水印

- Sony
- Leica
- Canon
- Fujifilm
- Nikon
- Xiaomi
- Apple
- Samsung
- OnePlus
- Vivo
